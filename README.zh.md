## 快速开始

浏览 [providers.json](data/providers.json) 查找免费 LLM API。每个提供商包含：

- **Base URL**：兼容 OpenAI 格式的 API 地址
- **注册地址**：获取 API Key 的链接
- **免费套餐**：免费类型与有效期
- **注册要求**：是否需要信用卡或验证
