## Schnellstart

Durchsuchen Sie [providers.json](data/providers.json) für kostenlose LLM-APIs.

- **Base URL**: OpenAI-kompatibler API-Endpunkt
- **Signup URL**: Link zum Erhalten von API-Schlüsseln
- **Kostenlose Stufe**: Art und Dauer des kostenlosen Zugangs
- **Anforderungen**: Kreditkarte oder Verifizierung erforderlich
