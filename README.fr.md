## Démarrage rapide

Consultez [providers.json](data/providers.json) pour trouver des APIs LLM gratuites.

- **Base URL** : Point de terminaison API compatible OpenAI
- **Signup URL** : Lien pour obtenir des clés API
- **Niveau gratuit** : Type et durée de l'accès gratuit
- **Exigences** : Carte de crédit ou vérification nécessaire
