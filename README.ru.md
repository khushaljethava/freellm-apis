## Обзор

Этот репозиторий поддерживает актуальную информацию о бесплатных LLM API:

- Детали поставщика и конечные точки API
- Ограничения бесплатного уровня и суммы кредитов
- Требования к регистрации (кредитная карта / верификация телефона)
- Даты последней проверки

**90+ поставщиков · Глобальные, Индия, Китай, Япония, Корея, Европа, Ближний Восток, ЮВА**

## Быстрый старт

Просмотрите [providers.json](data/providers.json) для поиска бесплатных LLM API.

- **Base URL**: API-эндпоинт, совместимый с OpenAI
- **Signup URL**: Ссылка для получения API-ключей
- **Бесплатный уровень**: Тип и продолжительность бесплатного доступа
- **Требования**: Нужна ли кредитная карта или верификация
