## अवलोकन

यह रिपोज़िटरी मुफ़्त LLM API की अद्यतन जानकारी रखती है:

- प्रदाता विवरण और API एंडपॉइंट
- फ्री टियर की सीमाएँ और क्रेडिट राशि
- साइनअप आवश्यकताएँ (क्रेडिट कार्ड / फ़ोन वेरिफ़िकेशन)
- अंतिम सत्यापन तिथि

**90+ प्रदाता · वैश्विक, भारत, चीन, जापान, कोरिया, यूरोप, मध्य पूर्व, दक्षिण-पूर्व एशिया**

---

## त्वरित शुरुआत

मुफ़्त LLM API खोजने के लिए [providers.json](data/providers.json) देखें।

- **Base URL**: OpenAI-संगत API एंडपॉइंट
- **Signup URL**: API Key प्राप्त करने का लिंक
- **Free Tier**: फ्री एक्सेस का प्रकार और अवधि
- **आवश्यकताएँ**: क्रेडिट कार्ड या वेरिफ़िकेशन की ज़रूरत
