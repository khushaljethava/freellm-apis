## Início rápido

Consulte [providers.json](data/providers.json) para encontrar APIs LLM gratuitas.

- **Base URL**: Endpoint de API compatível com OpenAI
- **Signup URL**: Link para obter chaves de API
- **Nível gratuito**: Tipo e duração do acesso gratuito
- **Requisitos**: Cartão de crédito ou verificação necessária
