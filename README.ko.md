## 개요

이 저장소는 무료 LLM API에 대한 최신 정보를 유지합니다:

- 제공업체 세부 정보 및 API 엔드포인트
- 무료 티어 제한 및 크레딧 금액
- 가입 요건 (신용카드 / 전화번호 인증 여부)
- 최종 확인 날짜

**90개 이상의 제공업체 · 글로벌, 인도, 중국, 일본, 한국, 유럽, 중동, 동남아시아**

---

## 빠른 시작

[providers.json](data/providers.json)을 참조하세요. 각 제공업체에는 다음이 포함됩니다:

- **Base URL**: OpenAI 호환 API 엔드포인트
- **Signup URL**: API 키를 얻는 링크
- **무료 티어**: 무료 액세스 유형 및 기간
- **요건**: 신용카드 또는 인증 필요 여부
