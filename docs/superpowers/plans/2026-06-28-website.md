# freellm-apis Website Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a fast, mobile-friendly, AdSense-ready Astro static site in `web/` that reads `data/providers.json` and `data/models.json` at build time and serves a homepage, per-provider pages, and a model browser.

**Architecture:** Astro static site (`output: "static"`) in `web/` subfolder. Data is imported directly from `../data/*.json` at build time — no fetching. Three page types: homepage with filter tables, dynamic provider detail pages, and a model browser. All styled with a single `global.css`, no framework.

**Tech Stack:** Astro 4, `@astrojs/sitemap`, vanilla JS (filter only), plain CSS.

## Global Constraints

- Astro version: `^4.0.0`
- `output: "static"` — no SSR, no server
- Data paths relative to `web/src/pages/`: `../../../data/providers.json` etc (adjust per file depth)
- No CSS framework — plain CSS only in `web/src/styles/global.css`
- No JS framework — vanilla JS only, inline in Astro `<script>` tags
- Font stack: `-apple-system, BlinkMacSystemFont, "Segoe UI", sans-serif`
- Max content width: `1100px`
- AdSense publisher ID via env var `PUBLIC_ADSENSE_ID` (placeholder `ca-pub-XXXXXXXXXX` if not set)
- No git push
- Vercel root dir: `web/`, build command: `npm run build`, output: `dist`

<div class="filter-bar" id="filter-bar">
  <button class="filter-btn active" data-filter="all">All</button>
  <button class="filter-btn" data-filter="permanent">Permanent Free</button>
  <button class="filter-btn" data-filter="credits">Free Credits</button>
  <button class="filter-btn" data-filter="no-card">No Card Required</button>

<script>
  const bar = document.getElementById('filter-bar');
  if (bar) {
    const btns = bar.querySelectorAll('.filter-btn');
    let activeFilter = 'all';
    let noCardOnly = false;

    function applyFilters() {
      document.querySelectorAll('tr[data-free-type]').forEach(row => {
        const r = row as HTMLTableRowElement;
        const matchType = activeFilter === 'all' || r.dataset.freeType === activeFilter;
        const matchCard = !noCardOnly || r.dataset.cardRequired === 'no';
        r.classList.toggle('hidden', !(matchType && matchCard));
      });
    }

    btns.forEach(btn => {
      btn.addEventListener('click', () => {
        const b = btn as HTMLButtonElement;
        if (b.dataset.filter === 'no-card') {
          noCardOnly = !noCardOnly;
          b.classList.toggle('active', noCardOnly);
        } else {
          activeFilter = b.dataset.filter || 'all';
          btns.forEach(x => { if ((x as HTMLButtonElement).dataset.filter !== 'no-card') x.classList.remove('active'); });
          b.classList.add('active');
        }
        applyFilters();
      });
    });
  }
</script>

  const provSel = document.getElementById('filter-provider');
  const modSel = document.getElementById('filter-modality');

  function applyFilters() {
    const prov = (provSel as HTMLSelectElement).value;
    const mod = (modSel as HTMLSelectElement).value;
    document.querySelectorAll('#models-tbody tr').forEach(row => {
      const r = row as HTMLTableRowElement;
      const matchProv = !prov || r.dataset.provider === prov;
      const matchMod = !mod || (r.dataset.modalities || '').split(',').includes(mod);
      r.classList.toggle('hidden', !(matchProv && matchMod));
    });
  }

  provSel?.addEventListener('change', applyFilters);
  modSel?.addEventListener('change', applyFilters);
</script>
